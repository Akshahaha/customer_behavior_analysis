# 🛍️ Customer Behavior Analysis

### Data Analytics Project using Python, SQL & Power BI

---

## 📌 Project Overview

This project analyzes customer shopping behavior to uncover key insights into purchasing patterns, customer segmentation, and revenue drivers.

Using a combination of **Python, SQL, and Power BI**, the project demonstrates an end-to-end data analytics workflow — from raw data processing to interactive dashboards and business recommendations.

---

## 🎯 Objectives

* Understand customer purchasing patterns
* Identify high-value customers and segments
* Analyze the impact of discounts and subscriptions
* Evaluate product and category performance
* Provide data-driven business recommendations

---

## 🧰 Tech Stack

| Tool             | Purpose                        |
| ---------------- | ------------------------------ |
| Python           | Data cleaning & preprocessing  |
| SQL (PostgreSQL) | Data storage & analysis        |
| Power BI         | Data visualization & dashboard |
| Jupyter Notebook | Exploratory Data Analysis      |

---

## 📂 Dataset

* Dataset: Customer Shopping Behavior
* Contains information such as:

  * Customer demographics (Age, Gender, Location)
  * Product details (Category, Item Purchased)
  * Purchase data (Amount, Frequency)
  * Behavioral data (Subscription, Discounts, Reviews)

---

## 🔄 Project Workflow

### 1️⃣ Data Collection

* Imported dataset from CSV into Jupyter Notebook

### 2️⃣ Data Cleaning (Python)

* Handled missing values
* Standardized column names
* Converted data types
* Removed inconsistencies

### 3️⃣ Data Storage (SQL)

* Loaded cleaned dataset into PostgreSQL
* Created structured tables for analysis

### 4️⃣ Data Analysis (SQL Queries)

Performed key business analysis such as:

* Revenue by gender and category
* Customer segmentation (New / Returning / Loyal)
* Impact of discounts on spending
* Subscription vs non-subscription behavior
* Top-performing products and categories

### 5️⃣ Data Visualization (Power BI)

Built an interactive dashboard including:

* KPI Cards (Total Revenue, Avg Spend, Customer Count)
* Sales trends
* Category performance
* Customer segmentation visuals
* Geographic insights

---

## 📊 Key Insights

* 💡 Subscribed customers tend to spend more on average
* 💡 A small number of products contribute to most revenue
* 💡 Discounts increase purchase frequency but not always revenue
* 💡 Loyal customers generate the highest long-term value
* 💡 Certain categories dominate overall sales
